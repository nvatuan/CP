This is a training contest based on my youtube tutorial on matrix exponentiation. After watching the video, try to solve at least first 6 problems without any help. Around 25th of July, I will post some hints and make a video or live stream with detailed solutions to all 9 problems.

Limak is always either happy or sad. His mood switches with probability pp every second. If Limak is happy right now, what is the probability that he's happy after nn seconds?

Input

An integer nn (1≤n≤1091≤n≤109) and a real value pp (0<p<10<p<1) with at most 9 digits after floating point.

Output

Print one line with the answer. The absolute error allowed is 10−610−6.

Examples
Input
Copy
1 0.7
Output
Copy
0.3000000000
Input
Copy
2 0.1
Output
Copy
0.8200000000
Input
Copy
11 0.06
Output
Copy
0.6225404294
Note

In the first sample test, there's probability 0.70.7 that Limak changes his mood from happy to sad. Otherwise (with probability 0.30.3), he stays happy.

In the second sample test, the answer is 0.1⋅0.1+0.9⋅0.9=0.820.1⋅0.1+0.9⋅0.9=0.82 because we want Limak either to switch his mood twice or not to change the mood at all.

Limak can be either happy or sad. His mood changes (or stays same) when he reads an English uppercase letter. Letters S and D always make him sad, H makes him happy, and every vowel (AEIOU) flips his mood. Other letters do nothing. Limak is happy right now and he wants to be happy after reading a string.

Let a valid character denote a question mark ? or an uppercase English letter A-Z.

You are given a pattern-string ss with nn valid characters. If there are, say, xx question marks, there are 26x26x ways to replace them with uppercase letters A-Z. Your task is to count ways of replacing so that Limak would be happy after reading a new string (if he's happy before).

Additionally, there are qq updates of form ,,i ci c", each modifying one character of the pattern ss. You need to print the answer before updates and after every update, modulo 109+7109+7.

Input

The first line contains two integers nn and qq (1≤n,q≤2000001≤n,q≤200000) — the length of the pattern and the number of updates.

The second line contains the initial pattern ss with nn valid characters.

Each of the following qq lines contains an integer ii (1≤i≤n1≤i≤n) and a valid character cc. The update changes the ii-th character of ss to cc. No update tries to change a character into the same character.

Characters in ss are 11-indexed. After you change a character, this change stays on (updates are not temporary).

Output

Print q+1q+1 lines, each with the current answer modulo 10000000071000000007.

Example
Input
Copy
2 5
A?
2 O
1 H
1 ?
2 ?
2 H
Output
Copy
6
1
0
7
403
26
Note

The pattern-string ss is initially A? and then after every next update: AO, HO, ?O, ??, ?H.

For the initial pattern A?, there are six good ways to replace all question marks: AA, AE, AH, AI, AO, AU. The first letter is a vowel so it flips Limak's mood from happy to sad. The second letter must be a vowel or H.
